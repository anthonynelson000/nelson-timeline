# Nelson Timeline

Interactive life-milestone timeline for Anthony + Victoria.
Live at https://nelsontimeline.netlify.app

## Editing milestones
Milestones are driven by a Google Sheet. Update the sheet, wait ~60s (Google CSV cache),
and reload the page.

Sheet: https://docs.google.com/spreadsheets/d/1Zi6QbElaSWjMCzry05IALHDLZgKajvVBHCWX4LiknPk

## Editing the design
Edit `index.html` and push to `main`. Netlify will auto-deploy.
